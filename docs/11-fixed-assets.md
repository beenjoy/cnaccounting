# 固定资产管理

本文档描述系统中固定资产的完整生命周期管理，遵循**中国企业会计准则 CAS 4《固定资产》**和 **CAS 8《资产减值》**，涵盖资产卡片、折旧计算、处置和减值处理。

---

## 1. 固定资产的定义与分类

### 定义

固定资产是指企业为生产商品、提供服务、出租或经营管理而持有的，使用寿命超过一个会计年度的有形资产。同时满足以下两个条件：
1. 与该固定资产有关的经济利益很可能流入企业
2. 该固定资产的成本能够可靠地计量

### 常见分类

| 分类 | 示例 |
|------|------|
| 房屋及建筑物 | 厂房、办公楼、仓库 |
| 机器设备 | 生产设备、仪器仪表 |
| 运输设备 | 汽车、叉车、船舶 |
| 电子设备 | 电脑、服务器、通信设备 |
| 办公设备及家具 | 桌椅、档案柜、复印机 |
| 其他固定资产 | 不属于以上分类的固定资产 |

**注：** 企业应根据自身行业和业务特点确定固定资产分类，并在会计政策中说明。

---

## 2. 固定资产卡片（台账）

每项固定资产在系统中建立一张**资产卡片**，记录该资产的完整信息：

### 基本信息

| 信息项 | 说明 |
|--------|------|
| 资产编号 | 系统内唯一标识（如 FA-2025-00001） |
| 资产名称 | 资产的描述性名称 |
| 资产分类 | 房屋建筑物/机器设备/运输设备等 |
| 所属公司 | 使用该资产的公司 |
| 使用部门 | 资产所在部门（影响折旧归属费用） |
| 存放地点 | 资产的实际存放位置 |
| 序列号/出厂编号 | 制造商的编号（可选） |
| 原值（入账成本） | 资产取得时的入账原值 |
| 净残值率 | 预计报废时残余价值占原值的百分比（如5%） |
| 净残值金额 | = 原值 × 净残值率 |
| 可折旧金额 | = 原值 - 净残值金额 |
| 折旧方法 | 直线法/双倍余额递减法/年数总和法/工作量法 |
| 预计使用年限 | 预计该资产能够使用的年数（年） |
| 月折旧额 | 每月应计提的折旧金额（自动计算） |
| 资产状态 | 在用、停用、已处置 |
| 入账日期 | 资产开始计提折旧的日期（通常为次月起） |
| 关联科目 | 固定资产原值科目和累计折旧科目 |
| 入账凭证 | 资产入账时对应的日记账凭证 |

### 累计数据（系统自动维护）

| 信息项 | 说明 |
|--------|------|
| 累计折旧 | 自入账起已计提的折旧合计 |
| 净值（账面价值）| = 原值 - 累计折旧 |
| 减值准备 | 已计提的固定资产减值准备 |
| 减值后账面价值 | = 净值 - 减值准备 |
| 已使用月数 | 自入账起已累计计提折旧的月数 |
| 剩余使用年限 | 预计使用年限 - 已使用月数（折算为年） |

## 3. 固定资产入账

### 原值的确定

固定资产应以**历史成本**入账，原值包括：

| 取得方式 | 原值构成 |
|---------|---------|
| 外购 | 购买价款 + 相关税费（不可抵扣增值税）+ 运输费 + 装卸费 + 安装费 + 专业人员服务费 |
| 自建 | 竣工前发生的必要支出（在建工程科目归集，完工后转入固定资产） |
| 融资租入 | 租赁开始日的公允价值与最低租赁付款额现值两者中较低者 |
| 债务重组取得 | 公允价值 |
| 非货币性资产交换取得 | 公允价值（满足商业实质条件时）或换出资产账面价值（不满足时） |

**注意：** 一般纳税人购入固定资产，可以抵扣的增值税进项税额不计入固定资产原值。

### 入账标准

固定资产单价标准由企业在会计政策中自行规定，低于该标准的有形资产通常作为低值易耗品或当期费用处理。典型标准：
- 制造业：单价 ≥ 2,000 元
- 服务业：单价 ≥ 500 元（企业自定）

### 入账会计分录

**外购固定资产（一般纳税人，税率13%）：**
- 借：固定资产（原值）100,000
- 借：应交税费——应交增值税（进项税额）13,000
- 贷：银行存款/应付账款 113,000

**在建工程转入固定资产（工程完工）：**
- 借：固定资产 X
- 贷：在建工程 X

**融资租入固定资产：**
- 借：固定资产（较低者） X
- 贷：长期应付款——融资租赁款 X

## 4. 折旧计算方法

固定资产应从**次月起**开始计提折旧，当月增加的固定资产当月不计提折旧。

### 4.1 直线法（年限平均法）——最常用

**计算公式：**
- 月折旧额 = （原值 - 净残值）÷ 预计使用年限 ÷ 12

**示例：**
- 设备原值：120,000 元
- 净残值率：5%，净残值：6,000 元
- 可折旧金额：114,000 元
- 预计使用年限：5年
- **月折旧额：114,000 ÷ 5 ÷ 12 = 1,900 元/月**

### 4.2 双倍余额递减法——加速折旧

前期多折，后期少折。每期按期初账面净值（不扣残值）乘以双倍直线折旧率计算。最后两年改用直线法（以保证剩余折旧额能摊完到净残值）。

**计算公式：**
- 年折旧率 = 2 ÷ 预计使用年限 × 100%
- 年折旧额 = 期初账面净值 × 年折旧率
- 月折旧额 = 年折旧额 ÷ 12

**示例（5年，原值100,000，净残值5,000）：**

| 年份 | 期初净值 | 年折旧率 | 年折旧额 | 期末净值 |
|------|---------|---------|---------|---------|
| 第1年 | 100,000 | 40% | 40,000 | 60,000 |
| 第2年 | 60,000 | 40% | 24,000 | 36,000 |
| 第3年 | 36,000 | 40% | 14,400 | 21,600 |
| 第4年 | 21,600 | — | 8,300（改直线）| 13,300 |
| 第5年 | 13,300 | — | 8,300（改直线）| 5,000（净残值）|

### 4.3 年数总和法——加速折旧

**计算公式：**
- 年折旧率 = 剩余可使用年数 ÷ 年数总和 × 100%
- 年数总和 = n × (n+1) ÷ 2（n为预计使用年限）
- 年折旧额 = （原值 - 净残值）× 年折旧率

**示例（5年，可折旧金额95,000）：**
- 年数总和 = 5+4+3+2+1 = 15

| 年份 | 剩余年数 | 折旧率 | 年折旧额 |
|------|---------|-------|---------|
| 第1年 | 5 | 5/15=33.3% | 31,667 |
| 第2年 | 4 | 4/15=26.7% | 25,333 |
| 第3年 | 3 | 3/15=20.0% | 19,000 |
| 第4年 | 2 | 2/15=13.3% | 12,667 |
| 第5年 | 1 | 1/15=6.7%  | 6,333 |

### 4.4 工作量法——按实际使用量

适用于使用程度变化较大的资产（如车辆按行驶里程，机器按生产量）。

**计算公式：**
- 单位工作量折旧额 = （原值 - 净残值）÷ 预计总工作量
- 月折旧额 = 本月实际工作量 × 单位工作量折旧额

**示例（卡车，原值300,000，净残值30,000，预计行驶100万公里）：**
- 每公里折旧额：270,000 ÷ 1,000,000 = 0.27 元/公里
- 某月行驶5,000公里，月折旧额：5,000 × 0.27 = 1,350 元

## 5. 折旧计提会计分录

每月末，根据各资产的折旧计算结果，自动或手动生成折旧凭证：

### 按使用部门归属折旧费用

| 使用部门 | 折旧科目 |
|---------|---------|
| 生产车间 | 制造费用——折旧费（最终进入主营业务成本） |
| 销售部门 | 销售费用——折旧费 |
| 管理部门 | 管理费用——折旧费 |
| 研发部门 | 研发费用——折旧费 |

**折旧分录（示例）：**

- 借：制造费用——折旧费 5,000
- 借：管理费用——折旧费 2,000
- 借：销售费用——折旧费 1,000
- 贷：累计折旧 8,000

## 6. 固定资产改良与后续支出

### 资本化 vs 费用化

| 支出类型 | 处理方式 | 判断依据 |
|---------|---------|---------|
| **资本化支出**（改良、扩建） | 计入固定资产原值，重新确定折旧 | 支出使资产产生了超出原估计标准的经济利益（如延长使用寿命、提升产能）|
| **费用化支出**（维修、保养） | 计入当期费用（管理费用或制造费用）| 仅维持原有性能，不带来额外经济利益 |

**资本化后续支出分录：**
- 借：固定资产（增加原值）X
- 贷：银行存款/应付账款 X
- （同时需重新计算剩余使用年限内的月折旧额）

## 7. 固定资产处置

### 7.1 出售固定资产

**第一步：注销固定资产**
- 借：固定资产清理（转入净值）= 原值 - 累计折旧
- 借：累计折旧（注销）
- 贷：固定资产（注销原值）

**第二步：若有减值准备，一并结转**
- 借：固定资产减值准备
- 贷：固定资产清理

**第三步：收到出售款项**
- 借：银行存款（含税售价）
- 贷：固定资产清理（不含税售价）
- 贷：应交税费——应交增值税（销项税额）（若适用）

**第四步：结转处置损益**

若出售价格 > 账面净值（盈利）：
- 借：固定资产清理
- 贷：资产处置收益

若出售价格 < 账面净值（亏损）：
- 借：资产处置损失（营业外支出）
- 贷：固定资产清理

### 7.2 报废固定资产

报废时通常无任何收入（或仅有残料收入）：

- 借：固定资产清理（净值）
- 借：累计折旧
- 贷：固定资产（原值）

收回残料：
- 借：原材料/银行存款 X
- 贷：固定资产清理 X

结转净损失（亏损）：
- 借：营业外支出
- 贷：固定资产清理

## 8. 固定资产减值（CAS 8）

### 减值迹象

在资产负债表日（年末），如存在以下迹象，须进行减值测试：

- 资产市价大幅下跌，跌幅超过正常使用耗损
- 资产的技术、市场、经济或法律环境发生重大不利变化
- 市场利率或其他投资收益率已提高，影响折现率
- 有证据表明资产已陈旧过时或实体损坏
- 资产的经济性能低于预期

### 减值测试

1. 确定资产的**可收回金额** = max（公允价值 - 处置费用，未来现金流量现值）
2. 若可收回金额 < 账面价值，则计提减值准备

### 减值会计分录

**计提减值准备：**
- 借：资产减值损失
- 贷：固定资产减值准备

**注意：** 固定资产一旦计提减值准备，后续**不得转回**（CAS 8 规定，与存货不同）。

### 减值后折旧调整

计提减值后，以**减值后的账面价值**作为新的折旧基础，按剩余使用年限重新计算折旧额：
- 新月折旧额 = （减值后账面价值 - 净残值）÷ 剩余使用年限（月）

---

## 9. 停用固定资产

**停用期间：**
- 停用的固定资产应**继续计提折旧**（除非该资产已完全折旧或划分为持有待售）
- 停用资产应在资产卡片中标记状态为"停用"
- 停用原因应在备注中说明

**重新启用：** 恢复使用时，更新资产状态为"在用"，继续按原折旧计划计提。

## 10. 税法折旧与会计折旧的差异

中国税法（企业所得税法）规定的折旧年限与会计上的预计使用年限可能不同：

| 资产类别 | 税法最低折旧年限 | 说明 |
|---------|--------------|------|
| 房屋、建筑物 | 20年 | 税法规定，不可缩短 |
| 飞机、火车、轮船、机器、机械及其他生产设备 | 10年 | 可申请加速折旧至5年 |
| 与生产经营有关的工具、器具、家具等 | 5年 | — |
| 飞机、火车、轮船以外的运输工具 | 4年 | — |
| 电子设备 | 3年 | 高新技术企业可申请2年 |

**会计 vs 税务差异处理（暂时性差异）：**
- 会计折旧 ≠ 税法折旧时，产生暂时性差异
- 暂时性差异须计算递延所得税（见 CAS 18 相关规则）

## 11. 固定资产台账报表

系统应能生成以下固定资产相关报表：

| 报表名称 | 内容 |
|---------|------|
| 固定资产明细台账 | 所有资产的原值、累计折旧、净值、本期折旧额 |
| 固定资产分类汇总表 | 按资产类别汇总原值、折旧和净值 |
| 本期新增/处置汇总 | 本期增加和减少的固定资产清单 |
| 折旧计划表 | 各资产未来各期的折旧预测 |
| 固定资产减值明细 | 已计提减值准备的资产清单及减值金额 |

# 增值税管理

本文档描述系统中增值税（VAT）的核心业务规则、科目结构、处理流程和申报要求，遵循中国现行增值税法规及**中国企业会计准则**。增值税是中国企业会计与西方会计最重要的实务差异，系统必须在凭证、科目和报告各层面全面支持。

---

## 1. 增值税纳税人分类

中国企业按年销售额和登记状态，分为两类增值税纳税人，两者在税率、发票开具和进项税额抵扣上有根本性差异：

| 类型 | 认定标准 | 可用税率 | 进项税额抵扣 | 发票类型 |
|------|---------|---------|------------|---------|
| **一般纳税人** | 年应税销售额超过500万元（工业/商业统一标准） | 13%、9%、6%、0% | ✅ 可据增值税专用发票全额抵扣 | 可开具增值税专用发票 |
| **小规模纳税人** | 年应税销售额未超过500万元 | 征收率3%（特定情形1%） | ❌ 不可抵扣进项税额 | 只能开具增值税普通发票（可申请税局代开专票） |

**业务规则：**
- 公司在系统中须配置增值税纳税人类型，该配置影响凭证录入时的税率选项和进项抵扣逻辑
- 一般纳税人一经认定，原则上不得转为小规模纳税人
- 新设公司在取得一般纳税人资格前，以小规模纳税人身份进行会计处理

---

## 2. 增值税税率体系

### 一般纳税人适用税率

| 税率 | 适用范围 |
|------|---------|
| **13%** | 销售货物（标准税率）；提供加工、修理修配劳务；进口货物 |
| **9%** | 销售粮食、食用植物油、农产品；提供交通运输、邮政服务；销售不动产；提供基础电信服务 |
| **6%** | 提供现代服务业（金融服务、信息技术、文化创意、咨询、广播影视等）；提供增值电信服务 |
| **0%** | 出口货物；跨境应税服务（如境外提供的咨询服务） |

### 小规模纳税人适用征收率

| 征收率 | 适用范围 |
|-------|---------|
| **3%** | 标准征收率（适用于绝大多数应税项目） |
| **5%** | 销售不动产、提供劳务派遣服务（特定情形） |

**注：** 增值税政策调整频繁，税率以国家税务总局当期有效规定为准。系统应支持税率配置，而非硬编码。

## 3. 增值税发票类型

发票类型决定了购买方能否抵扣进项税额，是中国增值税管理的核心：

| 发票类型 | 开具方 | 购买方能否抵扣 | 说明 |
|---------|-------|-------------|------|
| **增值税专用发票（专票）** | 一般纳税人 | ✅ 可抵扣（需认证） | 最重要的抵扣凭证，必须注明购买方税号、地址等完整信息 |
| **增值税普通发票（普票）** | 一般纳税人或小规模纳税人 | ❌ 不可抵扣 | 供个人消费者或小规模纳税人使用 |
| **电子增值税专用发票（电子专票）** | 一般纳税人（电子化） | ✅ 可抵扣 | 与纸质专票法律效力相同，2021年后全面推广 |
| **电子增值税普通发票（电子普票）** | 各类纳税人（电子化） | ❌ 不可抵扣 | 电子版普通发票 |
| **收购发票** | 特定企业收购农产品时自开 | ✅ 可按规定比例抵扣 | 计算抵扣，非专票抵扣 |
| **海关进口增值税专用缴款书** | 海关 | ✅ 可抵扣 | 进口货物缴纳增值税的凭证 |

## 4. 应交增值税科目结构

中国会计准则要求，一般纳税人使用**应交税费——应交增值税（2221）**科目，该科目下设多个专栏，详细记录进项税额、销项税额等：

### 一般纳税人科目结构

```
2221 应交税费
  └── 应交增值税（明细科目）
       ├── 借方专栏（增加抵扣或已缴税额）
       │    ├── 进项税额        → 取得合规专票时确认的可抵扣税额
       │    ├── 已交税金        → 本月预缴或补缴的增值税
       │    ├── 减免税款        → 税局批准的减免增值税
       │    └── 出口抵减内销应纳税额
       │
       └── 贷方专栏（增加应缴税额）
            ├── 销项税额        → 开具发票时确认的应缴税额
            ├── 进项税额转出    → 不可抵扣的进项税额转回费用
            ├── 出口退税        → 外贸企业出口退税额
            └── 转出未交增值税  → 月末将应纳税额结转至"未交增值税"

  └── 未交增值税（明细科目）
       → 月末结转的应补缴增值税（贷方），实际缴纳后冲销（借方）

  └── 待抵扣进项税额（明细科目）
       → 已取得专票但尚未认证或暂不符合抵扣条件的进项税额

  └── 增值税留抵税额（明细科目）
       → 当期进项税额大于销项税额时，多余的进项留待下期抵扣
```

### 小规模纳税人科目结构

小规模纳税人结构简单，使用单一科目：
```
2221 应交税费——应交增值税
  → 直接按含税价的征收率（3%）计算应纳税额，无进项抵扣明细
```

## 5. 进项税额抵扣规则

### 可抵扣条件（全部满足）

1. **取得合规抵扣凭证**：增值税专用发票、海关缴款书、收购发票等
2. **凭证内容与业务一致**：发票上的货物/服务与实际业务相符
3. **在规定时限内认证**：自发票开具之日起180天内完成电子认证（逾期不可抵扣）
4. **用途符合规定**：不用于不可抵扣情形（见下方）

### 不可抵扣情形

以下情形取得的进项税额**不得**从销项税额中抵扣：

| 不可抵扣情形 | 处理方式 |
|------------|---------|
| 用于集体福利或个人消费（如员工福利、业务招待） | 进项税额转出，计入相关费用成本 |
| 用于免税项目 | 进项税额转出，计入免税项目成本 |
| 用于简易计税项目 | 进项税额转出，计入相关成本 |
| 购入后发生非正常损失的货物 | 损失对应的进项税额转出，计入损失 |
| 购买餐饮服务 | 直接列为费用，无进项可抵扣 |
| 购买居民日常服务、娱乐服务 | 直接列为费用，无进项可抵扣 |

**进项税额转出会计处理：**
- 借：相关成本/费用科目（金额 = 该笔进项税额）
- 贷：应交税费——应交增值税（进项税额转出）

## 6. 标准会计分录

### 6.1 采购业务（含增值税）

**场景：一般纳税人购入原材料，取得增值税专用发票**

含税价格 = 原材料价格 + 增值税（13%）

| 借/贷 | 科目 | 金额 |
|------|------|------|
| 借 | 原材料（或库存商品） | 不含税价 |
| 借 | 应交税费——应交增值税（进项税额） | 发票税额 |
| 贷 | 应付账款（或银行存款） | 含税价合计 |

**示例：** 购入原材料 100,000 元（不含税），增值税税率 13%
- 借：原材料 100,000
- 借：应交税费——应交增值税（进项税额）13,000
- 贷：应付账款 113,000

---

**场景：采购服务（增值税税率 6%）**

- 借：管理费用/销售费用 100,000
- 借：应交税费——应交增值税（进项税额）6,000
- 贷：应付账款 106,000

---

**场景：小规模纳税人采购（无进项抵扣）**

小规模纳税人取得的增值税普通发票，税额直接计入成本：
- 借：原材料 113,000（含税全额）
- 贷：应付账款 113,000

### 6.2 销售业务（含增值税）

**场景：一般纳税人销售货物，开具增值税专用发票**

| 借/贷 | 科目 | 金额 |
|------|------|------|
| 借 | 应收账款（或银行存款） | 含税价合计 |
| 贷 | 主营业务收入 | 不含税价 |
| 贷 | 应交税费——应交增值税（销项税额） | 发票税额 |

**示例：** 销售商品 200,000 元（不含税），增值税税率 13%
- 借：应收账款 226,000
- 贷：主营业务收入 200,000
- 贷：应交税费——应交增值税（销项税额）26,000

---

**场景：小规模纳税人销售（征收率 3%）**

小规模纳税人按含税收入换算后计算应纳税额（含税价 ÷ 1.03 × 3%）：

- 借：应收账款 103,000
- 贷：主营业务收入 100,000
- 贷：应交税费——应交增值税 3,000

### 6.3 月末增值税结转

**场景：月末，应交增值税（进项税额）< 应交增值税（销项税额），即本期应补缴增值税**

计算：应纳税额 = 销项税额合计 - 进项税额合计 - 留抵税额

将应纳税额结转至"未交增值税"：
- 借：应交税费——应交增值税（转出未交增值税）X
- 贷：应交税费——未交增值税 X

**场景：月末，进项税额 > 销项税额，形成留抵**

本期进项大于销项，无需补缴，留抵税额结转下期：
- 系统账面自动体现：应交增值税借方余额即为留抵税额
- 转入留抵税额明细科目：
  - 借：应交税费——增值税留抵税额 X
  - 贷：应交税费——应交增值税（进项税额）X

### 6.4 缴纳增值税

**场景：次月申报期内（15日前）缴纳上月应纳增值税**

- 借：应交税费——未交增值税 X
- 贷：银行存款 X

### 6.5 进项税额转出

**场景：已抵扣的进项税额因用于不可抵扣用途，需要转出**

例：购入货物后，有一批用于员工福利（不可抵扣）：
- 借：应付职工薪酬（福利费）X（等于该部分进项税额）
- 贷：应交税费——应交增值税（进项税额转出）X

## 7. 增值税申报流程

### 申报周期

| 纳税人类型 | 申报周期 | 申报期限 |
|---------|---------|---------|
| 一般纳税人（月申报） | 按月申报 | 次月15日前 |
| 一般纳税人（季申报，特殊情形） | 按季申报 | 次季度首月15日前 |
| 小规模纳税人 | 按季申报（一般） | 次季度首月15日前 |

### 月末增值税处理步骤

1. **核对进项税额**
   - 汇总本月所有已认证的增值税专用发票进项税额
   - 核对是否存在需要转出的进项税额（不可抵扣用途）
   - 执行进项税额转出分录

2. **核对销项税额**
   - 汇总本月所有已开具发票的销项税额
   - 确认发票开具金额与账面收入一致

3. **计算应纳税额**
   - 应纳税额 = 当期销项税额 - 当期进项税额 - 期初留抵税额
   - 若 > 0：本期有应补缴税额
   - 若 ≤ 0：形成留抵，结转下期

4. **生成结转凭证**
   - 将应纳税额从"应交增值税"结转至"未交增值税"
   - 或将留抵税额转入"增值税留抵税额"

5. **申报并缴纳**
   - 次月15日前通过电子税务局完成纳税申报
   - 缴纳税款，冲销"未交增值税"

## 8. 增值税与其他税的关系

### 税金及附加

增值税本身不计入"税金及附加"（利润表项目），但与增值税相关的**附加税**需计入：

| 附加税 | 计算基础 | 税率 | 科目 |
|-------|---------|------|------|
| 城市维护建设税 | 实际缴纳的增值税额 | 市区7%、县镇5%、农村1% | 税金及附加（6051） |
| 教育费附加 | 实际缴纳的增值税额 | 3% | 税金及附加（6051） |
| 地方教育附加 | 实际缴纳的增值税额 | 2% | 税金及附加（6051） |

**计提附加税分录（月末）：**
- 借：税金及附加
- 贷：应交税费——应交城市维护建设税
- 贷：应交税费——应交教育费附加
- 贷：应交税费——应交地方教育附加

### 企业所得税

增值税为价外税，计算企业所得税时：
- 收入按不含税金额确认
- 成本按不含税金额列支（可抵扣进项税额不计入成本）
- 不可抵扣的进项税额计入成本/费用，可在所得税前扣除

## 9. 特殊业务处理

### 出口退税

一般纳税人出口货物，适用增值税零税率，已缴纳的进项税额可申请退还：

**基本流程：**
1. 出口时确认收入（以人民币金额），增值税税率为0%
2. 向税务机关申报出口退税
3. 确认退税资格后，对应进项税额转入"出口退税"专栏
4. 收到退税款时，冲销相关科目

**会计分录（出口零税率，申报退税）：**
- 借：应收出口退税款
- 贷：应交税费——应交增值税（出口退税）

### 混合销售与兼营

**混合销售（一次销售同时含货物和服务）：**
- 按主业决定增值税税率（从主性原则）
- 若主业为货物销售，全部按13%征收增值税

**兼营（不同期进行不同类型应税项目）：**
- 分别核算，分别适用各自税率
- 未分别核算的，从高适用税率

### 不动产和建筑服务

- 不动产（销售）：适用9%税率；特定情形可适用简易计税（5%）
- 建筑服务：适用9%税率；跨县市项目须在项目所在地预缴

## 10. 系统配置要求

为支持增值税处理，系统需要在以下层面进行配置：

### 公司级配置

| 配置项 | 说明 |
|-------|------|
| 增值税纳税人类型 | 一般纳税人 或 小规模纳税人 |
| 税务登记号（纳税人识别号） | 18位统一社会信用代码 |
| 默认适用税率 | 公司主营业务的默认增值税税率 |
| 申报周期 | 月度申报 或 季度申报 |
| 所在地区 | 影响城市维护建设税税率（市区/县镇/农村） |

### 科目配置

| 科目 | 说明 |
|------|------|
| 应交税费——应交增值税（进项税额）| 可抵扣进项税额专栏 |
| 应交税费——应交增值税（销项税额）| 应缴销项税额专栏 |
| 应交税费——应交增值税（进项税额转出）| 不可抵扣进项转出专栏 |
| 应交税费——应交增值税（转出未交增值税）| 月末结转专栏 |
| 应交税费——未交增值税 | 月末结转应补缴税额 |
| 应交税费——增值税留抵税额 | 留抵税额科目 |
| 税金及附加（6051） | 附加税费（城建税、教育费附加等）|

### 凭证录入支持

- 录入采购/销售凭证时，支持分离不含税金额和税额
- 支持选择适用税率（13%/9%/6%/0%/3%）
- 支持标注发票类型（专票/普票/无发票）
- 税额自动计算：税额 = 不含税金额 × 税率

---

*本文档描述了增值税处理的核心业务规则，是中国企业会计最重要的本地化要求。增值税政策变化频繁，应以国家税务总局当期有效规定为最终依据。*
